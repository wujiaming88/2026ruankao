## 🔴 高频必背（反复考、反复错）

### 等保五级名称
```
1级：用户自主保护级
2级：系统审计保护级
3级：安全标记保护级
4级：结构化保护级
5级：访问验证保护级 ← 最高级！
```

### PKI四组件
```
CA（认证中心）：颁发/管理证书 ← 核心
RA（注册中心）：审核用户身份 ← 前台受理
CRL（证书撤销列表）：记录已撤销证书
证书库：存储已颁发证书
```

### 架构风格分类
```
数据流：管道-过滤器、批处理
调用/返回：主程序/子程序、面向对象、分层
数据为中心：仓库、黑板 ← 黑板在这！
独立构件：进程通信、事件驱动/隐式调用
虚拟机：解释器、规则系统 ← 解释器在这！
```

### SOA三标准 → "描通发"
```
WSDL → 描述服务
SOAP → 通信传输
UDDI → 发现注册
```

### ATAM四属性 → "性安修可"
```
性能、安全性、可修改性、可用性
```

### DSSA三阶段 → "分设实"
```
分析 → 领域模型
设计 → DSSA（特定领域软件架构）
实现 → 构件
口诀："分→模，设→架，实→件"
```

## ⚠️ 设计模式易混对（必背判断口诀）

### 策略 vs 状态
```
主动选哪种方式 → 策略（选交通工具）
处于某状态行为自动变 → 状态（红绿灯）
```

### 适配器 vs 桥接
```
接口不兼容/第三方接入 → 适配器（转接头，事后补救）
两个维度独立变化 → 桥接（遥控器×电视，事前设计）
```

### 建造者 vs 模板方法
```
分步造一个复杂对象 → 建造者（组装电脑）
定义算法骨架子类重写步骤 → 模板方法（做菜食谱）
终极判断：得到东西→建造者；执行过程→模板方法
```

### 命令 vs 备忘录
```
存动作/操作为对象 → 命令（录像倒放）
存状态快照 → 备忘录（游戏存档）
终极口诀：命令存"动作"，备忘录存"状态"
```

### 外观 vs 代理 vs 中介者
```
多个子系统统一简化入口（外→内） → 外观（酒店前台）
控制单个对象访问（一→一） → 代理（保镖）
多个模块相互协调（内↔内） → 中介者（群聊管理员）
```

### 享元 vs 单例
```
全局只有1个实例 → 单例
按类型共享少量几个实例（节省内存） → 享元
```

### 工厂方法 vs 抽象工厂
```
创建一种产品 → 工厂方法
创建一族/一套产品 → 抽象工厂
```

## 📝 其他必背

### V模型测试对应
```
单元测试 ← 详细设计
集成测试 ← 概要设计
系统测试 ← 需求规格说明书
验收测试 ← 用户需求
```

### 规范化
```
1NF：列原子（不可再分）
2NF：消除部分函数依赖（非主属性完全依赖主键）
3NF：消除传递函数依赖
BCNF：每个决定因素都是候选键
```

### 奈奎斯特定理
```
采样频率 ≥ 2 × 信号最高频率
```
