## 🔴 高频必背（反复考、反复错）

### 等保五级名称
```
1级：用户自主保护级
2级：系统审计保护级
3级：安全标记保护级
4级：结构化保护级
5级：访问验证保护级 ← 最高级！
```

### PKI四组件
```
CA（认证中心）：颁发/管理证书 ← 核心
RA（注册中心）：审核用户身份 ← 前台受理
CRL（证书撤销列表）：记录已撤销证书
证书库：存储已颁发证书
```

### 架构风格分类
```
数据流：管道-过滤器、批处理
调用/返回：主程序/子程序、面向对象、分层
数据为中心：仓库、黑板 ← 黑板在这！
独立构件：进程通信、事件驱动/隐式调用
虚拟机：解释器、规则系统 ← 解释器在这！
```

### SOA三标准 → "描通发"
```
WSDL → 描述服务
SOAP → 通信传输
UDDI → 发现注册
```

### ATAM四属性 → "性安修可"
```
性能、安全性、可修改性、可用性
```

### DSSA三阶段 → "分设实"
```
分析 → 领域模型
设计 → DSSA（特定领域软件架构）
实现 → 构件
口诀："分→模，设→架，实→件"
```

## ⚠️ 设计模式易混对（必背判断口诀）

### 策略 vs 状态
```
主动选哪种方式 → 策略（选交通工具）
处于某状态行为自动变 → 状态（红绿灯）
```

### 适配器 vs 桥接
```
接口不兼容/第三方接入 → 适配器（转接头，事后补救）
两个维度独立变化 → 桥接（遥控器×电视，事前设计）
```

### 建造者 vs 模板方法
```
分步造一个复杂对象 → 建造者（组装电脑）
定义算法骨架子类重写步骤 → 模板方法（做菜食谱）
终极判断：得到东西→建造者；执行过程→模板方法
```

### 命令 vs 备忘录
```
存动作/操作为对象 → 命令（录像倒放）
存状态快照 → 备忘录（游戏存档）
终极口诀：命令存"动作"，备忘录存"状态"
```

### 外观 vs 代理 vs 中介者
```
多个子系统统一简化入口（外→内） → 外观（酒店前台）
控制单个对象访问（一→一） → 代理（保镖）
多个模块相互协调（内↔内） → 中介者（群聊管理员）
```

### 享元 vs 单例
```
全局只有1个实例 → 单例
按类型共享少量几个实例（节省内存） → 享元
```

### 工厂方法 vs 抽象工厂
```
创建一种产品 → 工厂方法
创建一族/一套产品 → 抽象工厂
```

```

```

## 🔴 网络OSI层次（4/8~4/9新增，硬伤区！）

### OSI七层 + 设备 + 协议 + 数据单元
```
7 应用层    | 网关      | HTTP/FTP/SMTP/DNS        | 数据/报文
6 表示层    | —        | 格式转换/加密/压缩           | 数据
5 会话层    | —        | 建立/管理/终止会话           | 数据
4 传输层    | —        | TCP/UDP                  | 段(Segment)
3 网络层    | 路由器    | IP/ICMP/ARP/OSPF         | 包(Packet)
2 数据链路层 | 交换机    | PPP/HDLC                 | 帧(Frame)
1 物理层    | 集线器    | —                        | 比特(Bit)
```

### 各层寻址方式
```
链路层 → MAC地址
网络层 → IP地址
传输层 → 端口号
```

### 地址解析四协议
```
ARP  = IP → MAC（地址解析）
RARP = MAC → IP（反向解析）
DNS  = 域名 → IP
DHCP = 动态分配IP
```

### 防火墙三类型对应层
```
包过滤     → 网络层（第3层）
状态检测   → 传输层（第4层）
应用代理   → 应用层（第7层）
口诀："包网状传应应"
```

### 攻击方式辨析
```
Spoofing = 伪造身份/IP，冒充（假快递员）
MITM     = 中间窃听+篡改（偷拆信件）
DoS      = 大量请求瘫痪（堵门）
Brute Force = 穷举密码（挨个试钥匙）
```

### 子网计算公式
```
可用主机数 = 2^主机位 - 2（减网络地址+广播地址）
```

### VLAN
```
VLAN = 逻辑隔离广播域（缩小，不是增大！）
同VLAN = 二层通信
跨VLAN = 必须经过三层（路由器/三层交换机）
```

### TCP vs UDP
```
TCP：可靠、有序、慢（网页、邮件、文件）
UDP：不可靠、无序、快（视频、游戏、DNS）
```

### SSL/TLS vs SSH vs IPSec
```
SSL/TLS → Web安全（HTTPS = HTTP + SSL/TLS）
SSH     → 远程登录安全
IPSec   → 网络层VPN加密
```

## 🔴 大数据+AI+区块链（4/8新增）

### 数据仓库 vs 数据湖
```
仓库 = 写时模式（Schema on Write）→ 清洗后 → 质量高
数据湖 = 读时模式（Schema on Read）→ 原始数据 → 质量低
```

### 四大神经网络
```
CNN = 图像（卷积、空间特征）
RNN = 序列（文本、语音、时间序列）
GAN = 生成（生成图片、换脸、对抗）
Transformer = 大模型（GPT、BERT、注意力机制）
```

### 大数据计算框架延迟排序
```
MapReduce：分钟~小时（批处理）
Hive：分钟级（SQL查询，底层MR）
Spark：秒级（内存微批）
Flink：毫秒级（真正流处理）← 毫秒选Flink！
```

### Hadoop三核心
```
HDFS = 存数据（分布式文件系统）
YARN = 管资源（调度框架）
MapReduce = 算数据（计算框架）
```

### Lambda vs Kappa
```
Lambda = 批处理 + 流处理两条路并行
Kappa  = 只用流处理一条路
```

### 区块链四核心
```
共识机制：PoW(拼算力,慢) / PoS(拼持币,快) / PBFT(投票,联盟链)
智能合约：自动执行的合约代码
哈希算法：保证不可篡改
数字签名：验证交易者身份
```

### 微服务事务
```
微服务场景 → Saga模式（最终一致性、补偿机制）
分布式数据库 → 2PC两阶段提交（强一致性、全局锁）
```
