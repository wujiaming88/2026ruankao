## 总体策略

1. **论文先行** — 你有真实项目经验，论文素材从第一周就开始准备

## 阶段一：基础构建 + 论文素材（第1-3周，3/17 - 4/6）

### 目标
- 搭建知识框架，覆盖核心考点
- 完成论文项目素材整理（2-3个项目）
- 做1套真题摸底

### 每日安排

| 时段 | 工作日（1.5-2h） | 周末（3-4h） |
|------|-------------------|--------------|
| 第1部分 | 知识点学习（1h） | 知识点学习（1.5h） |
| 第2部分 | 论文素材/真题（0.5-1h） | 真题练习（1.5-2h） |

### 第1周（3/17-3/23）— 架构核心
- [x] 📚 **架构风格**（必考！~5题）：管道-过滤器、分层、MVC、微服务、SOA、事件驱动、微内核 ✅ 3/17完成
- [ ] 📚 **质量属性 & 战术**：可用性、性能、安全性、可修改性、可测试性、易用性
- [ ] 📚 **ATAM评估方法**：9个步骤、效用树、风险点/非风险点/敏感点/权衡点
- [ ] 📝 **论文**：整理第1个项目素材（项目名、角色、背景、技术方案、效果）
- [ ] 🏋️ 周末做1套综合知识真题（2024年或2025年），摸底定位

### 第2周（3/24-3/30）— 设计模式 + 软件工程
- [ ] 📚 **23种设计模式**（必考！~3题）：重点记忆名称、意图、适用场景
  - 创建型5种 → 结构型7种 → 行为型11种
- [ ] 📚 **UML建模**：用例图、类图、序列图、状态图、组件图
- [ ] 📚 **需求工程**：功能需求/非功能需求、需求获取方法
- [ ] 📝 **论文**：整理第2个项目素材
- [ ] 🏋️ 做1套案例分析真题，熟悉题型

### 第3周（3/31-4/6）— 信息安全 + 数据库 + 网络
- [ ] 📚 **信息安全**（必考5-8题）：对称/非对称加密、数字签名、PKI、访问控制、WPDRRC模型
- [ ] 📚 **数据库**：关系设计、规范化（1NF-BCNF）、NoSQL、Redis缓存、数据仓库
- [ ] 📚 **计算机网络**：OSI/TCP模型、网络安全、VPN、防火墙
- [ ] 📝 **论文**：整理第3个项目素材（备用）
- [ ] 🏋️ 做第2套综合知识真题

- [ ] 🏋️ 做第3套综合知识真题 + 错题整理
- [ ] 📝 写第1篇论文初稿（微服务架构设计 ⭐⭐⭐⭐⭐ 热门方向）

### 第5周（4/14-4/20）— 案例分析专项
- [ ] 📚 **案例答题模板**整理：

  - 数据库题：ER图→关系模式→规范化

- [ ] 🏋️ 做3套案例分析真题（2022-2025），总结答题套路

### 第6周（4/21-4/27）— 操作系统 + 数学 + 法律 + 英语

- [ ] 📚 **法律法规**：知识产权、著作权、专利权（~3-4题，背就能拿分）

- [ ] 🏋️ 做第4套综合知识真题
- [ ] 📝 **论文修改**：打磨2篇论文，控制字数（摘要300-330字+正文2000-2500字）
